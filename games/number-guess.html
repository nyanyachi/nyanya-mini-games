About Number Guess

Number Guess chooses a hidden number in the supported range and asks you to find it by submitting guesses.

Each hint helps narrow the possible answer. The goal is to find the correct number using as few valid attempts as possible.

How to Play

  1. Review the displayed range of 1-100.
  2. Enter a number in the guess field.
  3. Press the Guess button or submit the form.
  4. Use the Too High or Too Low message to narrow the range.
  5. Keep guessing until the message says Correct!
  6. Start a new round when you are ready for another hidden number.

Number Range and Target

The hidden number is an integer from 1 through 100, including both endpoints.

A target number is generated when the page loads and again when New Game is pressed. The target stays the same until the round is completed or restarted.

Understanding Attempts

The attempt counter increases after a submitted guess passes the game's range check. A single repeated valid guess still counts as another attempt.

The final attempt total is the number of valid guesses used to find the target. Fewer attempts are better because Best stores the lowest completed attempt count.

Hints and Feedback

Too High means your guess is greater than the hidden number. Too Low means your guess is less than the hidden number.

Correct! appears when the guess equals the hidden number. After that, the input is disabled and further submissions show a message asking you to start a new game. Previous guesses are not listed on the page.

Input Rules

The field is a required number input with a displayed range of 1 to 100. The script also checks for guesses below 1 or above 100 and shows Enter a number between 1 and 100 without counting an attempt.

Browser number-input behavior may block empty or non-numeric entries before the game script runs. Decimal values inside the range can be submitted by the browser, but the hidden target is always a whole number.

Tips for Fewer Guesses

  • Start near the middle of the remaining range.
  • Use every Too High or Too Low clue to remove part of the range.
  • Keep track of the lowest and highest possible values after each guess.
  • Avoid repeating a guess unless you intentionally want to check it again.
  • Use a steady narrowing approach instead of choosing numbers randomly.

Controls and Devices

Use the number input to enter a guess, then press the Guess button. Because the guess area is a form, pressing Enter while the input is active may also submit the guess in supporting browsers.

The game can be played with mouse, touch, or keyboard input on desktop and mobile browsers.

Saving and Restarting

Number Guess saves only the Best score in this browser using localStorage. Best is the fewest valid attempts used to complete a round.

New Game creates a new target, clears the current attempt count and message, enables the input, and keeps the saved Best score. Reset Statistics removes the saved Best score but does not start a new round by itself. Clearing browser storage may also remove the saved record.
