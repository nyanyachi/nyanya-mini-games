About Color Guess

Color Guess asks you to study a target color swatch and choose the matching color from a small set of visual options.

Each round is a short comparison challenge. The game uses color swatches rather than color names, RGB text, or HEX codes.

How to Play

  1. Look at the Target Color swatch.
  2. Compare it with the three color choices.
  3. Select the choice that matches the target swatch.
  4. Read the Correct! or Try again! feedback.
  5. Press New Round when you want a fresh set of colors.
  6. Try to raise your score and saved best score over repeated rounds.

Understanding the Color Prompt

The prompt is a visible target swatch. You compare that swatch with three answer buttons that are also displayed as colors.

The game does not ask you to type a color name or interpret a visible RGB or HEX value. Spelling and capitalization do not matter because answers are chosen by selecting a color button.

Choices and Round Generation

Each round creates three random RGB colors. One of those colors is always the correct answer because it is also used as the target swatch.

The answer positions are shuffled before they appear. New Round creates a new set of colors, and similar colors may appear again in later rounds because colors are generated randomly.

Correct and Incorrect Answers

A correct selection adds 1 point to your score, may update Best Score, and shows Correct! as feedback.

An incorrect selection shows Try again! and still counts as an attempt. After any selection, the round is marked answered, so the same round cannot be answered again until you press New Round.

Understanding Results

Score is the number of correct answers in the current session. Total Attempts is the number of rounds where you selected an answer.

Best Score is the highest score saved in this browser. Higher score and best score values are better. The game does not track streaks or timed rounds.

Tips for Better Color Matching

  • Compare the overall hue first before focusing on brightness.
  • Check whether choices differ mainly in lightness or saturation.
  • Look at all three choices before selecting.
  • Use the same screen when comparing repeated results.
  • Do not rely on the previous answer position, because choices are shuffled each round.

Controls and Accessibility

You can choose colors with a mouse click or by tapping on a touch screen. The choices are standard buttons, so browsers may also allow keyboard activation when a choice is focused.

Color perception, screen brightness, and display settings can affect how close the choices appear. This game is a casual visual matching activity, not a color-vision test.

Saving and Restarting

Color Guess saves only Best Score in this browser using localStorage. The current score, attempts, and active round are not saved separately.

New Round creates a fresh target and three choices while preserving the current score, attempts, and saved best score. Reset Statistics sets score, attempts, and best score back to 0 and removes the saved best score. Clearing browser storage may also remove the saved record.
