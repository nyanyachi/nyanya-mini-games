About Typing Speed Test

Typing Speed Test measures how quickly and accurately you type a short displayed text sample.

The goal is to type the shown text, finish an exact match, and review your time, WPM, accuracy, and best saved WPM across short practice rounds.

How to Play

  1. Review the target text shown on the page.
  2. Press Start Test when you are ready to begin.
  3. Type the displayed text into the text area.
  4. Match the target text exactly to complete the test.
  5. Review your WPM, accuracy, time, and best WPM.
  6. Use New Text for another prompt when you want a fresh round.

Understanding WPM

WPM stands for words per minute. In this game, WPM is calculated from the number of typed characters divided by 5, then adjusted for the elapsed time in minutes.

The timer starts when you press Start Test and stops when your typed text exactly matches the target text. A higher WPM means the completed round was typed faster by this calculation.

Understanding Accuracy

Accuracy compares each typed character with the character in the same position in the target text. Exact spelling, spacing, punctuation, and capitalization all matter because the characters must match.

The percentage is based on the larger of the typed length and target length, so missing or extra characters can lower accuracy. Correcting mistakes before completion can improve the final accuracy because the result is calculated from the current text.

Text Matching and Completion

The test completes only when the typed text exactly matches the full target text. Extra characters, missing characters, different spacing, punctuation changes, or capitalization differences will prevent completion.

If the text still contains an error when you reach the end of the prompt, the timer keeps running until the text is corrected to an exact match.

Tips for Better Practice

  • Focus on accuracy first, then work on increasing speed.
  • Keep a steady rhythm instead of rushing through the prompt.
  • Use the same device and keyboard when comparing repeated results.
  • Review punctuation, spacing, and capitalization before restarting.
  • Try several rounds instead of relying on one score.

Controls and Devices

Use a physical keyboard or an on-screen keyboard to type in the text area after pressing Start Test.

Start Test begins a timed round, New Text loads the next built-in prompt and resets the current round, and Reset Statistics clears the saved best WPM.

Saving and Resetting

Typing Speed Test saves only your best WPM in this browser using localStorage. Time, current WPM, accuracy, and typed text are reset during new rounds and are not saved separately.

New Text changes to the next prompt, stops any active timer, clears the text area, and keeps the saved best WPM. Reset Statistics removes the saved best WPM and resets the current round. Clearing browser storage may also remove the saved best WPM.
