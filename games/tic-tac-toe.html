About Tic-Tac-Toe

Tic-Tac-Toe uses a three-by-three board where two local players place X and O in empty spaces.

The goal is to complete a horizontal, vertical, or diagonal line before the other player. This version is player versus player on the same device, not a computer-opponent game.

How to Play

  1. Start with a fresh board or press New Game to begin another round.
  2. Player X selects an empty square first.
  3. Player O takes the next turn, then turns continue alternating.
  4. Keep selecting empty squares until X wins, O wins, or the board fills.
  5. Review the result message and saved totals.
  6. Press New Game to clear the board for another round.

Board and Turn Rules

The board contains nine spaces arranged as three rows and three columns. Player X uses X, Player O uses O, and X always moves first in a new game.

Occupied squares cannot be selected again. Turns alternate automatically after each valid move. There is no computer move or difficulty setting in this version.

Winning and Drawing

A win requires three matching symbols in one line. Horizontal rows, vertical columns, and the two diagonals all count.

The round becomes a draw when every square is filled without a winning line. After a win or draw, the board is locked and no more spaces can be selected until New Game is pressed.

Understanding the Results

Current Player shows whose turn it is during an active round. The message below the board shows the current turn, X wins!, O wins!, or Draw! when the round ends.

X Wins, O Wins, and Draws count completed rounds saved in this browser. The game does not track total rounds separately, streaks, score, or match history.

Strategy Tips

  • Look for a winning move before choosing a defensive move.
  • Block the other player when they already have two symbols in a line.
  • The center square connects to more winning lines than an edge square.
  • Corners can help create multiple possible lines.
  • Check rows, columns, and diagonals before selecting a space.

Opponent Behavior

This is a local two-player game. Players share the same device and alternate turns manually.

There is no computer opponent, random move selection, blocking logic, winning-line strategy, or difficulty level.

Controls and Accessibility

You can select spaces with a mouse click or by tapping on a touch screen. The board cells and action controls are standard buttons, so browsers may also allow keyboard activation when a button is focused.

Saving and Restarting

Tic-Tac-Toe saves X Wins, O Wins, and Draws in this browser using localStorage. The current board does not survive a page refresh.

New Game clears the board, sets the current player back to X, unlocks the board, and keeps the saved totals. Reset Statistics removes the saved totals and returns X Wins, O Wins, and Draws to 0 without starting a new board by itself. Clearing browser storage may also remove saved records.
